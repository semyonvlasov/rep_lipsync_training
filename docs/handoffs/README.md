# Handoffs

This directory is for moving working context between chats and accounts.

Recommended format:
- one handoff file per thread/topic
- date in the filename
- short scope in the filename
- newest file can link back to older related handoffs

Suggested naming:
- `YYYY-MM-DD-topic.md`
- `YYYY-MM-DD-topic-followup.md`

Suggested contents:
- current goal
- what was changed
- current runtime status
- important files and commands
- known risks or open questions
- copy-paste prompt for the next chat

It is fine to keep several handoffs from different conversations here at the same time.
The easiest workflow is to point the next chat at the specific handoff file you want it to continue from.
